# Changelog

All notable changes to Tabula will be documented in this file.

## [0.1.0] - 2025-01-XX

### Added
- Initial release of Tabula programming language
- Whitespace-structured syntax (TAB for scope, SPACE for flow)
- Lexer with whitespace tokenization
- Parser with AST generation
- Code generation (native and WASM targets)
- Interpreter for testing
- Language Server Protocol (LSP) support
- Package Manager (tabpm)
- Interactive REPL
- Test framework
- Debugger with breakpoints
- Type checker with inference
- Code linter
- Performance profiler
- Documentation generator
- Standard library modules (IO, Strings, Collections, HTTP, Async, Math)
- Comprehensive tooling suite
- VS Code integration
- Makefile for common tasks

### Features
- Compile to native binaries
- Compile to WebAssembly
- Format code automatically
- Run programs interpretively
- IDE integration via LSP
- Package management
- Interactive development
- Testing support
- Debugging capabilities
- Code quality checks
- Performance analysis
- Auto-generated documentation
